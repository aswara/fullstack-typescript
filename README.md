# Fullstack Typescript Nest JS and Next JS ⚡️

## Backend
- Nest JS
- Sequilize
- Express JS
- Swagger
- Helmet

```
cd backend/
create file .env copy from .env.example
npm install
docker-compose -f docker-compose.db.yaml up -d
npm run migrate
npm run start:watch
```

## Frontend
- React JS
- Next JS
- Material UI
- Formik
- Axios
- Atomic Design

```
cd frontend/
npm install
npm run dev
```
